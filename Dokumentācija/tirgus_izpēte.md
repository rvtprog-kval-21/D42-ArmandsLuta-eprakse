# Tirgus izpēte
Tirgū nav pieejams nekādas citas web aplikācijas, vai aplikācijas, kas veic tādās pašas funkcijas kā eprakse.

Tuvāki konkorenti varētu būt prakse.lv, kur var meklēt prakses, un mācibu sistēmas kā mykoob.

Bet Eprakse ļāuj praktikantiem atrast savas prakses vietas vēlajamos uzņēmumos, un atvieglo dienasgrāmatu veidošanu.
Uzņēmumiem atvieglo praktikantu meklēšanu, un atzīmju likšana, praktikantu vērtēšana.
Skolām attiecigi atvieglo prakšu menedžēšanu, praktikantu pārskatu, atskaišu veidošanu.
