# Eprakses
Izveidot elektronisko prakses gramatu. Kur uznemumi var izveidot prakses vietas. Prakses vadītaji var likt atzimes par praksi. Skolas admin var apskatit, izveidot prakses un piesaisit skolas vaditaju, skolenam.Prakses vaditajs var apskatit vinam piesaistitos skolenus un to panakumus prakse. Skolens var apskatit visas vina ievaditas prakses vecas vai tagadejas.
